# Dodge the Balls

Simple game where you dodge moving balls, made with Lua and Löve2D.

## Installation

You need git and [Löve2D](https://love2d.org/) installed.

```sh
$ git clone https://github.com/vanillaiice/dtb-love2d
$ cd dtb-love2d
$ love .
# or
$ make
```

## Game Controls

`Move Mouse` - Move Ball

`Space` - Pause

`Backspace` - Restart

`Esc` - Quit
